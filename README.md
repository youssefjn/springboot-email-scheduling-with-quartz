
# springboot-email-scheduling-with-quartz

A Spring boot app to schedule Email with Quartz and 
MySQL as a database.


## API Reference

#### Schedule Email

``
  POST /scheduleEmail
``

| Body | Description                |
| :-------- | :------------------------- |
| `ScheduleEmailRequest` | **Required**. |



#### 


## Tech Stack

 Spring boot, MySQL
